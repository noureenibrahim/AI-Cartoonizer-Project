## Description
This project uses Computer Vision and Deep Learning to turn images and webcam feeds into cartoons.

## Features
* **Classical Mode:** Uses OpenCV (Bilateral Filter + Adaptive Thresholding).
* **AI Mode:** Uses a GAN (Generative Adversarial Network) trained on the Paprika style, optimized with K-Means clustering for cleaner colors.

## How to Run
1. Install requirements: `pip install opencv-python torch torchvision streamlit`
2. Run the app: `streamlit run main.py`
